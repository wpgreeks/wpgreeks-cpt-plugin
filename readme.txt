== Installation ==

= Installation method one via search =
1. Visit the Add New plugin screen and search for "WPGreeks CPT".
2. Click the "Install Now" button.
3. Activate the plugin.
4. Navigate to the "WPGreeks CPT" Menu.

= Installation method two via zip =
1. Visit the Add New plugin screen and click the "Upload Plugin" button.
2. Click the "Browse..." button and select zip file from your computer.
3. Click "Install Now" button.
4. Once done uploading, activate WPGreeks CPT Plugin.

= Installation method three via Manually Process =
1. Upload the WPGreeks CPT folder to the plugins directory in your WordPress installation.
2. Activate the plugin.
3. Navigate to the "WPGreeks CPT" Menu.

That's it! Now you can easily start creating custom post types in WordPress.
